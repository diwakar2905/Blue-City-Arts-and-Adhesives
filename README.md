# Blue City Arts and Adhesives

> Crafting Spaces with Heritage and Precision

A premium multi-page website showcasing handcrafted furniture, artistic handicrafts, and high-performance adhesive solutions crafted in Rajasthan and delivered worldwide.

## Overview

**Blue City Arts and Adhesives** is a luxury brand specializing in three core offerings:

- 🪑 **Premium Furniture** - Handcrafted pieces with Rajasthani heritage designs
- 🎨 **Artistic Handicrafts** - Traditional block-printed and decorative art pieces
- 🏭 **Industrial Adhesive Solutions** - High-performance adhesives for manufacturing and construction

Our products are designed for residential, commercial, and hospitality projects with an emphasis on quality, durability, and timeless design.

## Getting Started

1. **Open in Browser**
   - Open `index.html` in a web browser to view the site

2. **Local Server (Optional)**
   - For better performance with images and media, serve locally:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (with http-server)
   npx http-server
   ```
   Then visit `http://localhost:8000`

3. **Customization**
   - Edit HTML files to update content
   - Modify `style.css` for design changes
   - Update `script.js` for functionality enhancements
